HTML & CSS

Webページの作成に使用されるげんごです。HTMLとCSSを組み合わせる事で、静的なページを作り上げることができます

jQuery

素敵な雨後きを手軽に実装できるJavaScriptライブラリです。アニメーション効果をつけたり、Ajax使って外部ファイルを読み込んだりと色々な事ができます。

Ruby

オープンソースの動的なプログラミング言語で、シンプルさと高い生産性を備えています。大きなWebアプリケーションから小さな日用ツールまで様々なソフトウェアを作ることができます。

PHP

HTMLだけではページの内容をかえることはできません。PHPはHTMLにプログラムを埋め込み　それを可能にします。
